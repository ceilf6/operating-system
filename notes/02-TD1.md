> 以 sandbox 中 02-...sh 为准

mkdir 默认无法”透层“，当父层级不存在时会报错

-p 选项配置之后就可以了

```bash
mkdir $(pwd)/test-files/dir1/dir2
# mkdir: /Users/a86198/Desktop/Lab/operating-system-app/sandbox/test-files/dir1: No such file or directory
# mkdir 默认无法”透层“，当父层级不存在时会报错
mkdir -p $(pwd)/test-files/dir1/dir2
# -p 选项配置之后就可以了
```

```bash

    echo "$input" >> $filePath

awk 'NR==3' $filePath

awk 'NR>=2 && NR<=5' $filePath

### $ 是打印的最后一行
sed -n '$p' $filePath
awk 'END{print}' $filePath

## 管道化配合读取第 2 行到 5 行的单词数
sed -n '2,5p' $filePath | wc -w

touch $dirPath/debut.txt
```
